		Takes a user document and new password, generates a salt, and updates the  password for that user document. You can optionally have the function send  the updated document to the server as well.





Tags:





  

    return:  you set $upload to false, then just the updated document  is returned. If you set $upload to true, then the result of Sag->put() is  returned, so you get the updated document and HTTP information.
  

  

    access:  public
  









	
        
Parameters:

    

    

          

        object  
        $doc  
        The user document. Expected to look like what  SagUserUtils->getUser() returns.
      

          

        string  
        $newPassword  
        The new password for the user.
      

          

        bool  
        $upload  
        Whether to PUT the document to the server after  updating it. Defaults to false.
