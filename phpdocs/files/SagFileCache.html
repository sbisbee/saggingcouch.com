\SagFileCache

Package: Cache
Cache to the local hard disk. Uses /tmp by default, but you can specify another location.
Cache keys are used for file names and the contents are JSON. System file sizes are used to calculate the cache's current size. Every operation goes to the disk, so this cache will not be ideal for high I/O operations.
