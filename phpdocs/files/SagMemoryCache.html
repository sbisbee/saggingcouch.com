\SagMemoryCache

Package: Cache
Stores cached items in PHP's memory as serialized JSON, which was benchmarked as being faster than serliaze() and clone.
Cache sizing is not supported with this caching mechanism. This is because PHP is not accurate at reporting memory allocation and it does not make sense to increase latency to implement a broken feature.
