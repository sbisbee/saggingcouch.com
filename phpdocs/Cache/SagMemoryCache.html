Returns the total size of the items in the cache in bytes. Not reliable if you're using SagMemoryCache.
SagCache::makeKey()
Generates the hash of the provided URL that will be used as the cache key.
SagCache::mayCache()
Returns whether or not the item may be cached. It has to be a stdClass that Sag would return, with a valid E-Tag, and no cache headers that tell us to not cache.
SagCache::remove()
Removes the item from the cache and returns it (null if nothing was cached).
SagCache::set()
Caches the item returned at the provided key, replacing any pre-existing cached item. If the cache's size will be exceeded by caching the new item, then it will remove items from the cache until there is sufficient room.
SagCache::setSize()
Sets the max size of the cache in bytes.
