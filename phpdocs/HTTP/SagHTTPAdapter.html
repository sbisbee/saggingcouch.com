Class Overview




Provides a common interface for Sag to connect to CouchDB over HTTP,  allowing for different drivers to be used thereby controling your project's  dependencies.

Child classes:






SagNativeHTTPAdapter

	
Uses native PHP sockets to communicate with CouchDB. This means zero new  dependencies for your application.






SagCURLHTTPAdapter

	
Uses the PHP cURL bindings for HTTP communication with CouchDB. This gives  you more advanced features, like SSL supports, with the cost of an  additional dependency that your shared hosting environment might now have.
